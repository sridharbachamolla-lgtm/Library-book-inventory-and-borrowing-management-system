# Library Book Management System

## Overview
The Library Book Management System is a Python 3 command-line app that helps manage a small libraryâ€™s books using a CSV file for data storage. It enables easy addition, updating, searching, borrowing, returning, and deletion of books. This project provides a practical and efficient book inventory solution for classrooms and academic settings. #PythonProject #LibraryManagement #BookInventory

## Features
- Add new books with unique ID #AddBook
- View all books #ListBooks
- Search by ID or title #SearchBook
- Borrow & return books, track quantity #BorrowBook #ReturnBook
- Update book info or remove books #UpdateBook #DeleteBook
- Data is stored in `library.csv` automatically #CSVStorage
- Robust error handling (duplicate IDs, invalid inputs) #ErrorHandling

## Technologies & Tools Used
- Python 3.x #Python
- Built-in csv module #csv
- Built-in os module #os
- No external dependencies
